# Pokedex Platform

*Explore your favourite pokemon!*

## Tech Stacks
- Typescript
- Next.js
- Prisma + SQL
- tRPC (React Query)
- Material UI

### How to run the application

### clone the repo
```
git clone <repository url>

```

#### .env
```
DATABASE_URL = " "

```
### install packages
 ```
 npm install

 ```
 
### NOte: Make sure you change the url to development ex:http://localhost:3000 on the TRPC-Provider file in the provider directory.

 ### Run the application

 ```
 npm run dev

 ```
